GAL VALDMAN — אתר סטטי

פתיחה מקומית:
1. חלץ את קובץ ה-ZIP.
2. פתח את index.html בדפדפן.

מבנה:
- index.html — תוכן האתר
- styles.css — עיצוב ורספונסיביות
- script.js — תפריט מובייל ואנימציות
- assets/images — תמונות ולוגו

לפני פרסום:
- להוסיף המלצות אמיתיות ותמונות לקוחות רק באישורן.
- לבדוק שהמחירים והמסלולים עדיין עדכניים.
- אפשר להעלות את התיקייה ל-Netlify, GitHub Pages או כל אחסון סטטי.
